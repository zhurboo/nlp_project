# 任务定义

- 输入：故事标题
-  输出：5 个句子的短故事
- 参考论文：https://www.cs.jhu.edu/~npeng/papers/AAAI19-story-planning.pdf
- 方法不做限制，如果基于别的论文需要有所改进。

# 数据集

- 数据集：ROCStories
- 训练集大小：98,159
- 测试集大小：3,744
- 数据集链接：http://cs.rochester.edu/nlp/rocstories/

# 评测标准

作业会根据大家的报告，结果，开题，汇报情况综合考虑。本次作业大家需按上
述要求提供测试集输出，由助教统计测试集上的结果，检索评价标准是BLEU
scores。
